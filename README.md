# EVM Faucet App

### Uses:
- NodeJs
- Express
- EJS
- [Tailwindcss](https://tailwindcss.com/docs/installation)
- Ethersjs

![Demo](./statics/img/demo.png)

### Setup

1. `cp .env.example .env` and replace the variables with your custom ones.
2. `npm install`

### Run
`npm start` or `nodemon`

Open the webapp on your browser.
